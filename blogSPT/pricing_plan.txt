# BlogSPT 요금제 기획서

## 1. 기본 정책
- 프리미엄 요금제: 월 9,900원 또는 연 99,000원 (약 17% 할인)
- 체험 기간: 프리미엄 기능 14일 무료 체험
- 환불 정책: 결제 후 7일 이내 전액 환불 가능

## 2. 무료 기능 (Free Plan)

### 2.1 기본 체험단 관리
- 체험단 모집 정보 확인
- 기본 캘린더 뷰 (신청/선정/마감일 확인)
- 기본 알림 기능 (브라우저 알림)
- 수동 체험단 일정 등록

### 2.2 리뷰 작성 지원
- 기본 리뷰 템플릿 제공 (3개)
- 기본 미션 체크리스트
- 간단한 글자수 체크
- 기본 이미지 첨부 관리

### 2.3 데이터 관리
- 최근 1개월 체험단 이력 조회
- 기본 통계 (참여 횟수, 선정률)
- 수동 데이터 백업

## 3. 프리미엄 기능 (Premium Plan)

### 3.1 고급 체험단 관리
- 실시간 체험단 모집 알림
- AI 기반 맞춤 체험단 추천
- 자동 신청 예약 기능
- 상세 캘린더 관리 (일정 충돌 체크, 우선순위 설정)
- 관심 키워드 기반 자동 필터링

### 3.2 고급 리뷰 작성 지원
- AI 리뷰 작성 도우미
- 프리미엄 리뷰 템플릿 (20개+)
- 실시간 미션 체크 및 알림
- 이미지 최적화 및 워터마크 관리
- 맞춤법/문법 검사
- SEO 최적화 추천
- 개인 글쓰기 스타일 프로필 기능
  * 기존 리뷰 글 분석을 통한 스타일 학습
  * 사용자별 어투/문체 패턴 저장
  * 글쓰기 스타일 자동 적용
  * 다중 스타일 프로필 관리 (상황별 다른 스타일 저장 가능)
  * 스타일 프로필 내보내기/가져오기

### 3.3 고급 데이터 분석
- 전체 이력 무제한 조회
- 상세 통계 및 분석 리포트
- 자동 데이터 백업
- 체험단 성과 분석
- 선정 확률 예측

### 3.4 생산성 도구
- 일괄 신청 기능
- 자동 일정 최적화
- 리뷰 작성 진행도 트래킹
- 마감일 자동 알림
- 크로스 디바이스 동기화

## 4. 프리미엄 등급 시스템 (추후 도입 예정)

### 4.1 실버 (월 9,900원)
